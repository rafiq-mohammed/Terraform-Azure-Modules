
# User Guide

## 1. Resource Group
Sample values:

`rg_array = {
  DEUSNETRAFARG01 = "eastus"  
  DEUSSVCRAFARG01 = "eastus" 
}`
###
DEUSNETRAFARG01 - Naming Convention followed
- D = Dev
- EUS = eastus
- NET/SVC/STG = Network/Services/Storage
- RAF = RAFIQ (Project-name)
- ARG = Azure resource group
